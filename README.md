# azure-entra-id-iam-tenant-setup
Hands on IAM project demonstrating Azure Entra ID tenant creation, RBAC, identity lifecycle foundations, and zero trust access control

## Objectives 
This project demonstrates the creation and initial configuration of a secure Azure Entra ID (Azure Active Directory) tenant, serving as the foundation for the Identity and Access Management (IAM) in a cloud enterprise environment. The lab focuses on identity lifecycle preparation, access control, and security best practices aligned with Zero Trust principles.

## Tools 
- Azure Portal
- Azure Entra ID (Azure Active Directory)
- Microsoft Identity Platform
- RBAC (Role-Based Access Control)
- MFA & Conditional Access (baseline)

## IAM Concepts Demonstrated
- Identity Tenant Architecture
- Authentication vs authorization
- Identity lifecycle readiness
- Least Privilege Access
- Role Based Access Control (RBAC)
- Zero Trust identity foundation
